# nAnB

This is a game I made in high school.

## Running

1. Clone or download this repository to your computer:

```bash
git clone https://github.com/shihyousyu/nAnB.git
cd nAnB
````

2. Make sure you have Python 3 installed.

3. Install Pygame:

```bash
pip install pygame
```

4. Run the game:

```bash
python game.py
```
